# TradePilot Crypto Bot

A sleek React crypto trading bot dashboard with live token pricing from a free public market API, AI guided trading controls, and multi-exchange connection readiness.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Deploy

Railway can build this Vite app with `npm run build` and run it with `npm run start`.
